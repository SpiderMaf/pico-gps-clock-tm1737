# pico-gps-clock-tm1737

Port of Jeff Geerling's pico-time minirack code by SpiderMaf. This is largely the same code but converted to use a TM1637 display rather than the Adafruit one Jeff used.

## Dependencies

Requires the tm1737.py library by Mike Causer: https://github.com/mcauser/micropython-tm1637
